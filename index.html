cutler

Powerful, declarative settings management for your Mac, with speed.

cutler aims to simplify your macOS setup experience into an "almost" one-command procedure. Define your settings once, then easily apply, track, and revert changes across your system‚Äîthink of it as infrastructure-as-code for your Mac.

Installation

Recommended install:

$ curl -s https://raw.githubusercontent.com/hitblast/cutler/main/install.sh | bash

Or, use your favorite package manager:

Using brew:

$ brew install hitblast/tap/cutler

Using cargo:

$ cargo install cutler

Using mise:

$ mise use -g cargo:cutler
